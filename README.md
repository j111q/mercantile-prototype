# mercantile-prototype

Static mirror of [mercantile.wordpress.org](https://mercantile.wordpress.org/) being used as a scaffold for design prototype work during Automattic's April 2026 hackathon.

Not affiliated with or endorsed by the live Mercantile store. Not for public use. All trademarks, product imagery, and branding remain the property of their respective owners.

## Why this exists

On 2026-04-17, Matt Mullenweg posted an internal ask that mercantile.wordpress.org "should be excellent" and framed it as the kind of site Woo's tools should showcase. This repo is a starting point for exploring design improvements — not a shipping target.

## Status

- **Phase A — static mirror** (current): pixel-accurate snapshot captured via `wget --mirror`. Cart, checkout, and search don't work — the mirror is HTML only.
- **Phase B — redesigned flows** (in progress): replacing individual pages with new designs on top of the mirror.
- **Phase C — before/after pitch** (future): comparison page for stakeholder review.
